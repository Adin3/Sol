# *Sol*

Rhythm game made with opengl for the olcCodeJam2022

![](https://i.imgur.com/YOSd4Mv.png)

# *Setup*

Open `start.bat` to generate the solution

# *Controls*

-movement- <br />

D - left <br />
F - middle left <br />
J - middle right <br />
K - right <br />

-action- <br />

Enter - interact/continue <br />
Space - God Mode (Invincibility)
